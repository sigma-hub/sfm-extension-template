# Advanced Shell Fallback Template

Use this starter when one runtime is preferred but you still want a fallback path on other environments.

## Includes

- `sigma.binary.getPath()`
- `sigma.shell.run()`
- `sigma.ui.withProgress()`
- PowerShell fallback on Windows

## Usage

1. Copy this folder into a new repository.
2. Rename the extension metadata in `package.json`.
3. Run `npm install`.
4. Run `npm run build`.
