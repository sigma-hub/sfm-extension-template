# Progress And Context Template

Use this starter when you need commands that read the current Sigma context, inspect extension settings, or show cancellable progress.

## Includes

- `sigma.context.getCurrentPath()`
- `sigma.context.getSelectedEntries()`
- `sigma.settings.getAll()`
- `sigma.ui.withProgress()`

## Usage

1. Copy this folder into a new repository.
2. Rename the extension metadata in `package.json`.
3. Run `npm install`.
4. Run `npm run build`.
