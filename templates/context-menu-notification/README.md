# Context Menu Notification Template

Use this starter when you want quick right-click actions that work on the current selection.

## Includes

- `sigma.contextMenu.registerItem()`
- `sigma.ui.showNotification()`
- `sigma.ui.copyText()`

## Usage

1. Copy this folder into a new repository.
2. Rename the extension metadata in `package.json`.
3. Run `npm install`.
4. Run `npm run build`.
