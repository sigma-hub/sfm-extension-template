# Minimal Command Template

Use this starter when you want the smallest possible extension with one command, i18n, and a notification.

## Includes

- `sigma.commands.registerCommand()`
- `sigma.i18n.mergeFromPath('locales')`
- `sigma.ui.showNotification()`

## Usage

1. Copy this folder into a new repository.
2. Rename the extension metadata in `package.json`.
3. Run `npm install`.
4. Run `npm run build`.
