# PowerShell Runtime Template

Use this starter when your extension is Windows-only and needs to run PowerShell commands or scripts.

## Includes

- `sigma.shell.runWithProgress()`
- PowerShell command candidate fallback
- Structured JSON output from PowerShell

## Usage

1. Copy this folder into a new repository.
2. Rename the extension metadata in `package.json`.
3. Run `npm install`.
4. Run `npm run build`.
