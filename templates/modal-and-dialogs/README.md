# Modal And Dialogs Template

Use this starter when your extension needs to inspect selected files in a modal or collect a path through the native file picker.

## Includes

- `sigma.ui.createModal()`
- `sigma.dialog.openFile()`
- `sigma.contextMenu.registerItem()`

## Usage

1. Copy this folder into a new repository.
2. Rename the extension metadata in `package.json`.
3. Run `npm install`.
4. Run `npm run build`.
