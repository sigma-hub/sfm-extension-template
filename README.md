# Sigma File Manager Extension Templates

This repository contains different extension templates. You can use them as the base for your new extension.

## Templates

- `templates/minimal-command` - command + notification starter
- `templates/context-menu-notification` - context menu action starter
- `templates/modal-and-dialogs` - form modal and file dialog starter
- `templates/progress-and-context` - progress API and context starter
- `templates/http-api` - HTTP requests with form and list-detail modal demos
- `templates/binary-deno` - bundled Deno binary starter
- `templates/powershell-runtime` - PowerShell runtime starter
- `templates/advanced-shell-fallback` - shell fallback patterns
